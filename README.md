Hướng dẫn nhanh

- Mở file `index.html` trong trình duyệt (double-click hoặc kéo vào tab trình duyệt).
- Để hiển thị ảnh của bạn: đặt file ảnh vào `assets/photo.jpg` (kích thước càng lớn thì chất lượng càng tốt).
- Khi nhấp vào lá thư, lá thư sẽ mở và ảnh sẽ được tải lên với hiệu ứng mượt.

Tùy chỉnh
- Muốn đổi tên ảnh, chỉnh `data-src` ở `index.html` hoặc đổi `script.js` để lấy đường dẫn khác.
- Bạn có thể thay màu chủ đạo bằng cách chỉnh biến trong `:root` của `styles.css`.

Gợi ý: nếu bạn muốn host tĩnh, chỉ cần upload toàn bộ thư mục lên Netlify / GitHub Pages / Vercel.
